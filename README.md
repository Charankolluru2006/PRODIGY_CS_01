# Caesar Cipher Encryption Tool

## Overview
This project is a simple Caesar Cipher Encryption and Decryption tool developed using Python.

The Caesar Cipher is one of the oldest encryption techniques where each letter in the message is shifted by a fixed number of positions in the alphabet.

---

## Features
- Encrypt text messages
- Decrypt encrypted messages
- User-defined shift value
- Supports uppercase and lowercase letters
- Preserves spaces and symbols

---

## Technologies Used
- Python 3

---

## How It Works
The program takes:
1. A message from the user
2. A shift value
3. Choice for encryption or decryption

Example:
- Original Message: `hello`
- Shift Value: `3`
- Encrypted Message: `khoor`
